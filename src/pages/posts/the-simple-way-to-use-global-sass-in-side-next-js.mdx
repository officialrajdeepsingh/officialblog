### Import Sass

![Paste Code Inside _app.js](/images/post/2020/11/Paste-Code-And-Import.png "Paste Code Inside _app.js")

- Paste Code Inside Your `_app.js` File And Import Your scss File With Absolute
  Path and Restart your Application.

```javascript
// _app.js
import '../styles/global.scss';

// This default export is required in a new `pages/_app.js` file.
export default function MyApp({ Component, pageProps }) {
	return <Component {...pageProps} />;
}
```
