Man Command Provide the user manual Interface. This Interface Help To Read
Manual Page references of Any Command Inside Your Terminal.

### Syntax

```bash
man [OPTION]... [COMMANDS]...
```

**Example:**

```bash
man ls
```

**Result:**

![Output in your cmd Tool](/images/post/2021/01/screenshot-nimbus-capture-2021.01.08-01_15_26.png "Output in your cmd Tool")

- **Press q** inside your terminal. You Quit The Command And Return.
- **Press h** Read Summary of Commands

---

## Options

### `man -w`

This Flag Doesn’t display any manual pages but prints the locations Of Command,
Which Place Save Inside Your Laptop.

![Output man Command result](/images/post/2021/01/flgs.png "Output man Command result")

---

### `man -f` or `man --whatis`

This Flag Show You a short description from the manual page if a Description is
available.

![Output man Command option result](/images/post/2021/01/flag.png "Output man Command option result")

---

### `man man`

this way, You Read also man Command Manual Page Information.

![man Command Output](/images/post/2021/01/manualpage.png "man Command Output")

---

### `man --help`

This Flag Provides You help page document About man Command.

![man Command Output with --help flag](/images/post/2021/01/manhelpflags.png "man Command Output with --help flag")

---

### `man --usage`

This Flag gives you a short usage message about the Command.

![man Command Output with --usage flag](/images/post/2021/01/screenshot-nimbus-capture-2021.01.08-01_47_17-1.jpg "man Command Output with --usage flag")

---

### `man -V` or `man --version`

This Flag print man Command version In You Terminal.

![man command output version](/images/post/2021/01/MAN-version.png "man command output version")

---

### Note

- Man Command Provide Lots Off option We Recommend You Visit Command Man Page for Learn More
- Quit man Command simple use q for quit or Exit Command.use Keyboard arrow keys use For Scroll Page up for page up↑ and down for page down↓ keys.
