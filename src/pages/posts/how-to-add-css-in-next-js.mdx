## New Update

Next.js Version 9.3 **Support CSS Global Stylesheets.** Now you add CSS directly Import `.css` files as global stylesheets.

```javascript
import './style.css';
```

---

**Go To Github Download or Use Npm:**

```bash
npm install --save @zeit/next-css
or
yarn add @zeit/next-css
```

---

Create an `next.config.js` At the root of your project

## Default

default config use for import CSS Global stylesheet in custom _app.js

```javascript
const withCSS = require('@zeit/next-css');
module.exports = withCSS({});
```

## Custom

Custom config used for import CSS in other Components like header, footer like
so.

```javascript
const withCSS = require('@zeit/next-css');
module.exports = withCSS({
	cssModules: true, // After true than use import statement in next.js
});
```

---

## How To add Global CSS

When you create your app, help with npm. in the next step, you create a global
app. If you npm official command the by default app create in your project and import your Global CSS file in next.js custom _app.js

```javascript
import '../styles.css';
// or
import '../styles.scss';
// This default export is required in a new `pages/_app.js` file.
export default function MyApp({ Component, pageProps }) {
	return <Component {...pageProps} />;
}
```
