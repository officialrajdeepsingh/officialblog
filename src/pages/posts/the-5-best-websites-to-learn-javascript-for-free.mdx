JavaScript is the programming language of the web. You’re interested in building
web applications or becoming a web developer. JavaScript courses will help you even if you are an advanced
JavaScript programmer. Once considered a strictly client-side language, Node.js
allows JavaScript to run both the frontend and the backend. There are some
popular frontend JavaScript frameworks and libraries like Angular.js and
React.js.

### 1. [Mozilla’s For JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide)

The Mozilla JavaScript Guide hosted on their developer resources network is one
of the most platforms for javascript. It’s ideal for any skill level, and it
covers a wide variety of topics, from basic to more advanced javascript.

---

### 2. [W3schools](https://www.w3schools.com/js/)

W3school is a free online JavaScript course that you can take through W3schools
— they offer a wide variety of programming lessons. When you’re done, you can
take a final exam to earn a JavaScript certification. It’s a good place for a
beginner to intermediate coders. Experienced programmers might come across a
thing or two here, but there’s no sense in sorting through all the basic lessons
to locate something relevant.
