A sticky Note is a note-taken app containing row text and bullet points, and it comes with a basic text editor. The most important feature is To stick to the desktop screen and rearrange the Sticky Note app desktop.

![Demo](/images/stick-note-demo.gif "Demo")

## How to install Sticky App on gnome?

You can install [Sticky App with flatpak](https://flathub.org/apps/com.vixalien.sticky) in one command.

```bash
flatpak install flathub com.vixalien.sticky
```

![Install Sticky Note App in gnome distro](/images/stick-note.gif "Install Sticky Note App in gnome distro")

## How to increase productivity with Sticky Note App?

The Sticky Note App help with notes, to-do list and, most important, It helps with copying paste text or command.

You do not need to go back and copy a text and paste it once; you can create a list with Sticky App and then use it.
