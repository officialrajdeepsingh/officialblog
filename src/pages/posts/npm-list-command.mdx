The NPM list command helps list all the dependencies you install in the root
level project.

Npm list command print the Package list based on the package.json and
package.lock.json file.

```bash
npm ls
npm ls <option> or <flags>
```

![Npm ls command](/images/post/2022/03/carbon.png "Npm ls command")

NPM ls command is a similar command to the Linux ls command. But the difference
in npm and ls command. Linux ls command prints directory or folders, and the npm
ls command prints the installed packages in your project.

NPM ls command print package information with a beautiful design format in your
terminal.

We discuss only daily base options or flags in npm. npm ls command provides
many more flags, which many more flags work for the developer, and some commands
are not under everyday use.

We make sure with the npm ls command to speed up your workflow.

### Option or Flags

1. all
2. JSON
3. long
4. global
5. depth
6. link
7. package-lock-only

## all

npm ls --all print all the dependencies are depending upon the current project.

```bash
npm ls -all
```

### JSON

`npm ls --json` command print the data in json formate in the terminal.

```bash
npm ls json
```

![output the npm ls --json](/images/post/2022/04/npm-ls---json.png "output the npm ls --json")

### long

`npm ls --long` print the extended information.

```bash
npm ls --long
```

![output the npm ls --long](/images/post/2022/04/npm-ls---long.png "output the npm ls --long")

### global

npm ls global command prints all global installed package lists in the terminal.

```bash
npm ls --global
```

![output the npm ls --global](/images/post/2022/04/npm-ls---global.png "output the npm ls --global")

### depth

npm ls depth command prints in-depth dependencies into the terminal.

![output the npm ls --depth](/images/post/2022/04/npm-ls---depth.png "output the npm ls --depth")

### link

npm ls --link command only print the linked packages list in the terminal.

```bash
npm ls --link
```

![output the npm ls --link](/images/post/2022/04/npm-ls---link.png "output the npm ls --link")

### package-lock-only

npm ls package-lock-only command print dependence tree based on the `package-lock.json` file.

![output the npm ls --package-lock-only](/images/post/2022/04/npm-ls---package-lock-only.png "output the npm ls --package-lock-only")
