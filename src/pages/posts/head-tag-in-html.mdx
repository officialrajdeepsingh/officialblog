HTML Provide Lots Of tags. But Head Tag Define Web Page Information On Browser.
Head Tag Not be shown directly inside the Browser like Body Tag.In Head Tag, We
Define Title, Add CSS File, JS File, icons, and metadata.

![Head Tag In HTML Tags](/images/post/2020/12/head-tag-2.jpg 'Head Tag In HTML Tags')

- Always Use Head Tag In HTML Tags.

```html
<html> 

    <head> 
        <title> Title Here </title>
    </head>

</html>
```

---

**Syntax:**

```html
<head> 
	<title> Title Here </title>
</head>
```

### Example

```html
<!DOCTYPE html>
<html lang="en">
     <head>
         <meta charset="UTF-8">
         <title>Hello!</title>
     </head>
     <body>
         <h1>Hello World!</h1>
         <p>This is a simple paragraph.</p>
     </body>
</html>
```
