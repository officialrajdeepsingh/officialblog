**Basic Information About Fetch Method:**

The fetch() method is used to fetch a resource by API.

The request can be of any API that returns the data of the format JSON or XML.
This method returns a promise.

**Syntax:**

fetch( URL, Options )

**Parameters:** This method accepts two parameters.

- **URL:** First Is Url, API, And File Path.
- **Options:** It is an array of properties. It is an optional
  parameter.
